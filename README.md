# d.free

เว็บดาวน์โหลดวิดีโอแบบถูกลิขสิทธิ์สำหรับลิงก์ไฟล์วิดีโอโดยตรง เช่น `.mp4`, `.webm`, `.mov`, `.m4v` และ `.ogv`

## ขอบเขตการใช้งาน

- ใช้กับวิดีโอของคุณเอง
- ใช้กับไฟล์ตรงจากเซิร์ฟเวอร์หรือพื้นที่จัดเก็บที่คุณมีสิทธิ์
- ใช้กับวิดีโอที่เจ้าของอนุญาตให้ดาวน์โหลด
- ไม่รองรับการดึงไฟล์จาก YouTube, TikTok หรือ Instagram ของผู้อื่น

เปิดเว็บ แล้ววางลิงก์วิดีโอโดยตรงเพื่อสร้างปุ่มดาวน์โหลด

## ไฟล์ในโปรเจกต์

- `index.html` หน้าเว็บหลัก
- `styles.css` สไตล์ responsive
- `app.js` ตรวจสอบลิงก์และสร้างปุ่มดาวน์โหลด
